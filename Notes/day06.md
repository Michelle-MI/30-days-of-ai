# What I learned

- A dictionary store data as key-value pairs.
- Dictionaries use curly braces {}.
- Keys are used to access values instead of indexes.
- I can access a value using student["name"]
- I can update an existing value by assigning a new value to its key.
- I can add a new key-value pair by assigning a value to a new key.
- Dictionaries can store diffrent data types such as strings and integers.
- Dictionaries can be nested, meaning one dictionary can contain another dictionary.
- Dictionaries are useful for organizing related information, susch as a student's profile.
